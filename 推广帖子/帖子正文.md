## 截图指南

| 图 | 怎么截 |
|----|--------|
| 4.png | 浏览器打开 `192.168.1.1`，截 Argon 主题首页 |
| 5.png | LuCI → Services → Bandwidth Monitor |
| 6.png | LuCI → Statistics → Graphs，选一个好看的（CPU 或接口流量） |
| 7.png | 运行 `StressTest.ps1`，等跑完截最终报告 |
| 8.png | 拍路由器网口，最好有网线插着 |
| 10.png | LuCI → Services → Adblock，截域名数量页面 |
| 11.png | 浏览器打开 `jwgl.henu.edu.cn` 或 `lib.henu.edu.cn`，截教务系统页面 |
| 12.png | 路由器放桌上，旁边放个手机/电脑，WiFi 连着 |
