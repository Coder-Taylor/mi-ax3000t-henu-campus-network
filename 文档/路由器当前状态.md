## 存储结构

```
mtd0:  1MB   BL2          (启动引导)
mtd1:  256KB Nvram        (U-Boot 变量)
mtd2:  256KB Bdata        (系统变量)
mtd3:  2MB   Factory      (工厂数据/WiFi参数)
mtd4:  2MB   FIP          (固件映像)
mtd5-mtd7:    crash/KF    (日志区)
mtd8:  34MB  ubi_kernel   (内核+squashfs系统)
mtd9:  78MB  ubi          (可写overlay, ~58MB可用)
```

## 故障排查

| 问题 | 命令 |
|------|------|
| 认证失败 | `cat /tmp/campus_network.log` |
| 手动认证 | `/etc/campus_network/auto_login.sh` |
| WiFi断开 | `wifi reload` |
| 查看连接 | `iw dev phy1-sta0 link` |
| 查看IP | `ip addr show dev phy1-sta0` |
| 测试外网 | `ping 8.8.8.8` |
| 查看流控 | `tc -s qdisc show dev phy1-sta0` |
| 恢复出厂 | `firstboot` 然后重启 |
